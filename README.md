## ✨ Features

* ⚡ Multi-core CPU benchmarking using Web Workers
* 📊 Real-time charts with Chart.js
* 💻 Displays CPU & system information
* 🌙 Dark / Light mode toggle
* 📱 Fully responsive UI
* ⏳ Smooth progress bar & animations

## 🧠 How It Works

This tool runs parallel computations across CPU cores using Web Workers.
Each core processes tasks independently and returns a score.
All results are combined and visualized with charts for easy comparison.
