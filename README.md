# 🔥 Multi-Core CPU Benchmark

🚀 A modern web-based tool to test and visualize CPU performance across multiple cores in real time.

---

## ✨ Features

* ⚡ Multi-core benchmarking using Web Workers
* 📊 Interactive charts powered by Chart.js
* 💻 Displays CPU & system information
* 🌙 Dark Mode / Light Mode toggle
* 📱 Fully responsive design
* ⏳ Real-time progress bar and smooth animations

## 🛠 Technologies Used

* HTML, CSS, JavaScript
* Web Workers (multi-threading)
* Chart.js

---

## 📊 Description

This tool runs parallel tasks across CPU cores to measure performance. Each core processes calculations independently, and the results are combined into a total performance score with a visual chart.
